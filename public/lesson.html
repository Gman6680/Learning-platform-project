data types

Data types are the kinds of values that you can store and manipulate in JavaScript.

There are eight basic data types in JavaScript:


1. Number: Represents numeric data, including integers and floating-point numbers.

2. String: Represents textual data enclosed within single (' ') or double (" ") quotes.

3. Boolean: Represents logical values: true or false.

4. Undefined: Represents a variable that has been declared but not assigned a value.

5. Null: Represents the intentional absence of any object value.

6. Symbol: Represents unique identifiers. Introduced in ECMAScript 6 (ES6).

7. Object: Represents a collection of key-value pairs where values can be of any data type.

8. BigInt: Represents integers of arbitrary length. Introduced in ECMAScript 2020 (ES11).

basic syntax

In JavaScript, let's delve deeper into the basic data types mentioned in the previous lesson:

Number: Numeric values, such as integers or floating-point numbers.

Let's explore examples and use cases for each of these data types to understand their significance in JavaScript programming.
